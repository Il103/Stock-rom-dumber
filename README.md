# Stock ROM Dumber

> An intelligent GitHub Actions tool that uses **DumprX** to automatically dump and organize Stock ROMs

---

## Features (Smart Version)

| Feature | Description |
|---------|-------------|
| 🧠 **Smart Dumping** | Powered by DumprX — supports most Stock ROM formats (Recovery, Fastboot, etc.) |
| 📦 **Full Extraction** | Extracts all partitions + optional filesystem contents from images |
| 🌳 **Branch Mode** | Each ROM is uploaded to its own dedicated branch (e.g. `dump-xiaomi-spinel-...`) |
| ⚡ **High Speed** | aria2c multi-connection downloader + automatic disk cleanup |
| 🗜 **Optional Compression** | Converts `.img` files to `.img.xz` to save space |
| 🛡️ **Extra Intelligence** | Recovery fallback, skips large APKs (>100MB), Git LFS, retry logic |
| 📋 **Rich Info** | Auto-generated `_dump_info.txt`, SHA256 checksums, and file list |

---

## How to Use

### 1. Fork this repository
### 2. Run the Workflow
Go to: **Actions → Stock ROM Dumber → Run workflow**

#### Inputs:

| Input | Required | Description |
|------|----------|-------------|
| `rom_url` | Yes | Direct download link of the ROM |
| `target_branch` | No | Custom branch name (auto-generated by default) |
| `device_name` | No | Device codename (if auto detection fails) |
| `extract_contents` | No | Extract filesystem contents from images (default: true) |
| `compress_images` | No | Compress images to .xz (default: false) |

### 2. Download the ROM
After completion, go to **Branches** and switch to the newly created branch. All files will be in the root.

## Powered by DumprX

- Excellent support for HyperOS, MIUI, and various Stock ROMs.
- Handles payload.bin, sparse images, and different archive formats.
- Smart partition extraction.

---

## Requirements

- No secrets needed (uses default `GITHUB_TOKEN`).
- Runner: `ubuntu-latest`.
- For very large ROMs, enable `compress_images: true`.

---

## Notes

- Files are uploaded to **Branches** instead of Releases (better for large files).
- Git LFS is enabled automatically for big files.
- This is a **Smart Dumber** version developed by **mkpromvp**.
